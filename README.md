## 介绍

该项目基于Java开发。 

* 后端采用Spring Boot、Spring Security、Spring Data Jpa、Redis、Mysql。
* 前端采用Vue、Vant、Bootstrap。
* 本项目主要供交流学习，请不要用于商业项目。
